# TypeScript Learning Path — Starter Projects

Four scaffolded projects aligned with your learning roadmap.
Complete them in order — each builds on skills from the previous one.

## Projects

| # | Project | Folder | Skills |
|---|---------|--------|--------|
| 1 | **TS Utilities** | `01-ts-fundamentals/` | Types, interfaces, generics, async/await |
| 2 | **QA Automation Suite** | `02-playwright-qa/` | Playwright, test structure, assertions |
| 3 | **Stock Watchlist App** | `03-watchlist-app/` | React, Next.js, Express/Fastify, REST APIs |
| 4 | **AI Chat Assistant** | `04-ai-chatbot/` | Anthropic SDK, streaming, LangChain |

## Prerequisites

- [Node.js](https://nodejs.org/) v18+ installed
- A code editor (VS Code recommended — install the "Pretty TypeScript Errors" extension)
- `npm` or `pnpm` package manager

## Getting Started

Each project folder has its own `README.md` with:
- Setup instructions
- TODO challenges to complete
- Hints if you get stuck

Start with `01-ts-fundamentals/` after finishing the Total TypeScript course.

---

**Tip:** Run `npx tsc --noEmit` frequently to check for type errors without building.
